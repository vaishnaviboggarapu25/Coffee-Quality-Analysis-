# Global Coffee Quality Analysis Dashboard ☕

## Project Description
Designed an interactive Power BI dashboard to analyze global coffee quality metrics and defect trends using CQI data.

## Tools & Technologies
- Power BI
- Power Query
- DAX

## Data Preparation
- Cleaned and transformed sensory attributes such as Aroma, Acidity, Flavor, Body, and Aftertaste using Power Query.
- Handled missing values and standardized quality scores for analysis.

## Key KPIs (DAX Measures)
- Average Cup Points
- Defect Rates
- Top Coffee Origins
- Quality Score Distribution

## Visualizations
- Correlation analysis between quality attributes
- Geo-map showing coffee origin performance
- What-If analysis for quality scoring insights

## Key Insights
- Coffee processing methods significantly impact overall quality scores.
- Certain regions consistently produce higher-quality coffee with lower defect rates.
- Defects negatively affect final cup points and sustainability outcomes.
